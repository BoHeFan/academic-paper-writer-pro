# 论文内容智能生成工作流 (Automated Academic Content Generation Pipeline)

本 Skill 专用于 **Pipeline D**。详细规范请参考主 SKILL.md 文件。

## 功能概述

- 基于项目代码自动生成学术论文
- 支持多种输入格式（代码、笔记、实验数据）
- 内置学术诚信门禁系统
- 自动生成图表和公式

## 使用方式

通过主 SKILL.md 调用，或直接指定 Pipeline D。
