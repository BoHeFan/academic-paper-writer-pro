# 常见问题排查指南 (Troubleshooting Guide)

## 1. 安装问题

### 1.1 一键安装失败

**问题**：运行 \
px skills add\ 命令失败

**解决方案**：
1. 检查网络连接
2. 尝试使用手动克隆方式
3. 确保已安装 Node.js 和 npm

### 1.2 Skill 未被识别

**问题**：Agent 无法识别已安装的 Skill

**解决方案**：
1. 重启 Agent 会话
2. 显式要求 Agent "加载 academic-paper-writer-pro skill"
3. 检查 Skill 目录路径是否正确

---

## 2. 排版问题

### 2.1 格式错乱

**问题**：排版后的文档格式与预期不符

**解决方案**：
1. 确认提供了正确的模板文件
2. 检查 \config.json\ 中的 \ormat_params\ 设置
3. 尝试使用不同的格式预设

### 2.2 图片丢失

**问题**：排版后图片未正确嵌入

**解决方案**：
1. 确认图片文件存在于 \esources/figures/\ 目录
2. 检查图片路径是否正确
3. 确保图片格式为支持的格式（PNG、JPG等）

### 2.3 公式显示异常

**问题**：数学公式显示不正确

**解决方案**：
1. 检查 LaTeX 语法是否正确
2. 确认使用了正确的公式分隔符（\$...\$\ 或 \$$...\）
3. 查看 \esources/logs/\ 中的错误日志

---

## 3. 降AI率问题

### 3.1 技术术语被错误修改

**问题**：降AI率处理后技术术语被错误替换

**解决方案**：
1. 在 \esources/config.json\ 中添加术语保护列表
2. 手动检查并恢复被错误修改的术语
3. 使用更保守的处理策略

### 3.2 字数增加过多

**问题**：处理后字数超出原文太多

**解决方案**：
1. 检查 \config.json\ 中的扩展比例设置
2. 减少冗余扩展的程度
3. 删除不必要的修饰词

### 3.3 语义偏差

**问题**：处理后语义与原文不符

**解决方案**：
1. 检查替换词是否适合当前语境
2. 回退到原文，重新选择替换词
3. 使用更保守的处理策略

---

## 4. 性能问题

### 4.1 处理速度慢

**问题**：处理大型文档时速度很慢

**解决方案**：
1. 使用断点恢复功能，分段处理
2. 减少悬挂频率
3. 检查系统资源占用

### 4.2 内存不足

**问题**：处理大型项目时内存溢出

**解决方案**：
1. 使用延迟加载策略
2. 分批处理文档
3. 清理不必要的中间文件

---

## 5. 其他问题

### 5.1 中文乱码

**问题**：输出的中文文档出现乱码

**解决方案**：
1. 确保使用 UTF-8 编码
2. 检查字体设置
3. 尝试使用不同的输出格式

### 5.2 参考文献丢失

**问题**：参考文献未正确导入

**解决方案**：
1. 确认提供了 \.bib\ 文件
2. 检查 BibTeX 格式是否正确
3. 手动补充缺失的引用
